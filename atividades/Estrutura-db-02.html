Exercícios de Criação de Estruturas SQL

Exercício 1: Sistema de Biblioteca

Cenário: Você precisa criar um sistema para uma biblioteca municipal. Por enquanto, precisamos apenas registrar os livros.

Requisitos:

  • Database chamado biblioteca
  • Schema chamado acervo
  • Sequence para gerar números únicos para os livros
  • Tabela de livros com:
    • Código do livro (número)
    • Título do livro (texto, máximo 200 caracteres)
    • Ano de publicação (número)
  • Usuário do sistema chamado bibliotecario
