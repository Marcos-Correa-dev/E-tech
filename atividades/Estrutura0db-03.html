Exercícios de Criação de Estruturas SQL

Exercício 2: Sistema de Academia

Cenário: Uma academia precisa registrar seus equipamentos de musculação.

Requisitos:

  • Database chamado academia
  • Schema chamado inventario
  • Sequence para numeração dos equipamentos
  • Tabela de equipamentos com:
    • Número do equipamento (número)
    • Nome do equipamento (texto, máximo 100 caracteres)
    • Marca (texto, máximo 50 caracteres)
    • Data de aquisição (data)
  • Usuário do sistema chamado instrutor
