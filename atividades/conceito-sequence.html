Explicando Sequence de forma simples

Imagine que você tem uma lista de clientes e precisa dar um número único para cada um deles. A **sequência** é como um contador automático que faz isso para você.

Exemplo prático:

  • Quando você vai ao banco, pega uma senha: 001, 002, 003...
  • Quando vai ao açougue do mercado: 001, 002, 003...

Esses números nunca se repetem e sempre aumentam em 1.

No nosso caso:

create sequence cadastro.incrementador start 1 increment 1;

Vamos quebrar isso:

  • start 1: começa do número 1
  • increment 1: vai aumentando de 1 em 1

Então:

  • Primeiro cliente → pega número 1
  • Segundo cliente → pega número 2
  • Terceiro cliente → pega número 3
  • E assim por diante...

Por que usar?

  • Garante que cada cliente terá um número único
  • Você não precisa se preocupar em qual número dar para cada cliente
  • Evita números repetidos
  • O banco faz isso automaticamente para você

É como ter um funcionário dedicado só para distribuir números, mas de forma automática!
